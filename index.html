The difference between web 1.0,2.0,3.0:
Web 1.0
1) Web 1.0 used for sharing information. There was limited interaction between sites and the users, no opportunity to post reviews and feedback in the websites.

2) For queries in Web 1.0,if you had like to get information about "pills side effects",you had to insert the specific url addres of the website that posted information about that topic.
Web 2.0
1) Web 2.0 facilitates interaction between web users and sites.This changes allows users to interact more freely with each other. Web 2.0 encourages participation collaboration and information sharing.
In web 2.0 we got the option to socialize and interact with others through social media,blog and other internet website. Examples of web 2.0 applications:Youtube,Flicker,Facebook.

2) For queries in Web 2.0,you have to search the query "pills side effects" in the search engine and you will get results about all parts of the string that you typed in.
Web 3.0
1) The official name of Web 3.0 is Semantic Web.Web 3.0 is the executable phrase of world wide web with dynamic applications to access services and "machine to machine" interaction.
In web 3.0, computers can interpret information like human and intelligently generate and distribute useful content depend to the needs of the users(Films and series suggestions based on preference).

2) For queries in Web 3.0, you have to search the query "pills side effects" in search engine and you will get results about the exact topic and even related to your last explores
